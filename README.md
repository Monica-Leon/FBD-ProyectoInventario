# FBD-ProyectoInventario

Con este proyecto se busca crear un inventario del stock de libros de un Instituto con el fin de organizarlo de manera eficiente y sencilla. Contendrá el registro de libros, así cómo de los tutores y participantes. 

Este proyecto surge bajo la necesidad de llevar un orden y registro dento del instituto, para así evitar la pérdida de libros y el registro del avance de los participantes.
